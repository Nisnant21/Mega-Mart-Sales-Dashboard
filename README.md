 🛒 Mega Mart Sales Dashboard

 1. Project Title / Headline

**Mega Mart Sales Dashboard | Power BI**

---

 2. Short Description / Purpose

The **Mega Mart Sales Dashboard** is an interactive Power BI solution designed to analyze sales performance, customer behavior, and product profitability. It enables business stakeholders to monitor key sales metrics, identify trends, and make data-driven decisions to improve overall business performance.

---

 3. Tech Stack

* **Power BI Desktop** – Dashboard development and visualization
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – Calculated measures and KPIs
* **Data Modeling** – Building relationships between tables
* **Microsoft Excel / CSV** – Data source
* **Interactive Visualizations** – Charts, Maps, Slicers, and KPI Cards

---

 4. Data Source

The dashboard is built using a **Mega Mart retail sales dataset** containing transactional sales records from multiple product categories and regions. The dataset includes customer details, product information, sales transactions, profit, discounts, shipping details, and order dates. Data was imported into Power BI, cleaned using Power Query, and modeled for efficient reporting and analysis.

**Dataset includes:**

* Order ID
* Order Date
* Customer Name
* Product Name
* Category & Sub-Category
* Region / State / City
* Sales
* Profit
* Quantity
* Discount
* Shipping Mode

---

5. Features / Highlights

📊 Business Problem

Retail businesses generate large volumes of sales data, making it difficult to track revenue, profitability, customer purchasing patterns, and regional performance using traditional reporting methods.

 🎯 Goal of the Dashboard

To develop an interactive sales dashboard that provides real-time insights into sales performance, profit trends, customer behavior, and product performance, enabling better strategic and operational decision-making.

 🚀 Dashboard Features

* Executive KPI cards for Sales, Profit, Orders, and Quantity
* Region-wise and State-wise sales analysis
* Category and Sub-Category performance comparison
* Monthly and Yearly sales trend analysis
* Customer and product performance insights
* Profitability analysis across products and regions
* Interactive slicers for dynamic filtering
* Drill-down functionality for detailed exploration

 📈 Key Insights

* Identify top-performing products and categories
* Analyze regional sales and profit distribution
* Monitor monthly sales and profit trends
* Evaluate the impact of discounts on profitability
* Discover high-value customers and purchasing patterns
* Support data-driven business and inventory decisions
